Leer atentamente para un correcto funcionamiento
/************************************************
1.- Este código está pensado específicamente para linux, aunque también se puede
compilar en Windows, recomiendo usar TDM gcc (https://jmeubank.github.io/tdm-gcc/download/).
2.- Tener actualizado linux, usar comando "sudo apt-get update"
3.- Tener instalado gcc, usar comando "sudo apt install gcc"
4.- Para compilar el programa usar gcc:
4.1.- En la terminal, ir al directorio que contiene los códigos fuente
4.2.- escribir en la terminal el comando "gcc nombre_archivo.c -o nombre_archivo" (agregar .exe en windows)
4.3.- ahora que está compilado ejecutar el programa
5.- Escribir en la terminal el comando "./nombre_archivo" (nombre_archivo.exe en windows)
Nota: nombre_archivo se reemplaza con el nombre del archivo a compilar por ejemplo cola, pila, lista, etc.
